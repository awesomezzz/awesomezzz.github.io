# 關於 Hong's Blog

## 網站呈現與架設

應用 Github Page 與 Hexo 製作靜態網頁。

- Github Page 為靜態網頁託管服務。
- Hexo 為用來產生靜態網頁。


## 網站管理及功能

- 後台管理介面： Hexo Admin 。
- 留言板： Disqus 。
